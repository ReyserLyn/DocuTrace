# Backend - DocuTrace

## Descripción

Backend desarrollado en C++ utilizando el framework **Crow** para proporcionar una API REST que permite la búsqueda de documentos locales mediante un índice invertido.

## Características

- **API REST** con endpoints para búsqueda de documentos
- **Índice invertido** para búsquedas eficientes
- **Soporte múltiples formatos**: PDF, DOCX, TXT

## Tecnologías

- **C++17** - Lenguaje principal
- **Crow** - Framework web para C++
